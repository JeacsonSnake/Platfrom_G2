## 7. 问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 栈溢出 | LED 任务栈太小 | 从 2048 改为 4096 |
| LED 不亮 | 使用普通 GPIO 驱动 WS2812 | 改用 RMT + 自定义编码器 |
| WiFi 超时 | 网络环境不同 | 修改为当前可用 WiFi |
| MQTT 连接失败 | Windows 防火墙阻止 | 关闭防火墙或添加端口例外 |
| 网络不通 | NAT 网段隔离 | VMware 端口映射 1883→1883 |

## 8. 运行逻辑验证

### 启动流程
```
系统启动 → LED 初始化(白色闪烁3次) → WiFi 连接(黄色闪烁) 
→ 获取 IP → MQTT 连接(蓝色闪烁) → 连接成功(绿色常亮)
```

### 任务优先级
| 任务 | 优先级 | 说明 |
|------|--------|------|
| LED_TASK | 5 | 最高，确保状态实时更新 |
| MQTT_TASK | 1 | 网络通信 |
| 其他任务 | 1 | PWM/PCNT/PID |
