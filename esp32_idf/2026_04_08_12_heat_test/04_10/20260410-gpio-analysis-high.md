## 3. 推荐的高编号 GPIO (GPIO > 15)

### 3.1 首选推荐

| GPIO | 可用性 | 原因 | 推荐度 |
|------|--------|------|--------|
| **GPIO 21** | ✅ 完全可用 | 通用 GPIO，未引出特殊功能 | ⭐⭐⭐ 首选 |
| **GPIO 47** | ✅ 完全可用 | 通用 GPIO，与 48 相邻 | ⭐⭐⭐ 首选 |

### 3.2 次选推荐

| GPIO | 可用性 | 原因 | 推荐度 |
|------|--------|------|--------|
| **GPIO 22-25** | ✅ 可用 | 通用 GPIO，但需确认是否引出 | ⭐⭐ |
| **GPIO 33-37** | ✅ 可用 | 通用 GPIO，但需确认是否引出 | ⭐⭐ |
| **GPIO 38-42** | ✅ 可用 | 通用 GPIO，但需确认是否引出 | ⭐⭐ |
| **GPIO 45** | ✅ 可用 | 通用 GPIO | ⭐⭐ |

### 3.3 不推荐

| GPIO | 不可用原因 |

| GPIO 46 | Strapping pin（Boot 模式） |
| GPIO 48 | RGB LED 已用 |

## 4. 最终推荐配置

### 推荐 1: GPIO 21 (首选)

```
menuconfig:
    uart port tx pin: 21
```

**优势**:
- GPIO 21 在 ESP32-S3-DevKitC-1 上明确可用
- 远离其他已用 GPIO，避免干扰
- 位于开发板一侧，易于接线

**接线**:
```
USB-TTL RX → ESP32-S3 GPIO21 (TX)
USB-TTL TX → ESP32-S3 GPIO3  (RX)
USB-TTL GND → GND
```

### 推荐 2: GPIO 47 (备选)

```
menuconfig:
    uart port tx pin: 47
```

**优势**:
- GPIO 47 与 RGB LED (GPIO48) 相邻
- 明确可用，无特殊功能
